# 📸 Project Overview

HealMind AI is designed as an intelligent wellness assistant capable of understanding user emotions through facial expressions and conversational interaction.

The system combines:
- AI-based emotion recognition
- Voice interaction
- Generative AI responses
- Wellness-focused user experience

to create a more human-centered AI companion.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/nidhi-ai01/healmind-ai.git
cd healmind-ai

Frontend Setup
cd frontend
npm install
npm run dev

Frontend runs on:

http://localhost:3000
Backend Setup
cd backend
pip install -r requirements.txt
python main.py

Backend runs on:

http://localhost:8000
🔑 Environment Variables

Create a .env file inside the backend folder.

GROQ_API_KEY=your_api_key_here
